Network Analysis Algorithms for Disease Gene Prioritization

The compilation of protein-protein interaction data and the application of network traversal algorithms has been shown to be an effective approach to disease gene prioritization. In our project, we run three graph algorithms: PageRank, random walk with restart, and diffusion kernel. Each algorithm takes a set of known disease genes and uses this information to predict which other genes are associated with a given disease.
